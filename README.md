## Description

<!-- Présentation de ce qu'est ce site et résumé du projet en un paragraphe, toujours à jour-->

Symbiose est une installation intéractive démontrant une potion chimique virtuelle où chaque station contrôle un ingrédient (eau, flamme, poudres, tourbillon). Les participants devront faire face à différents événements aléatoire qui perturberont l'équilibre de la potion. Leurs but ultime est que la potion reste stable le plus longtemps possible.
